# L1 Overview - 生活健康基线

## When to use
- week starts or health baseline review day arrives
- two or more baseline metrics miss target for 3 days

## Inputs
- Input 1: sleep, activity, and meal data
- Input 2: weekly schedule

## Outputs
- Output 1: health baseline plan with targets
- Output 2: daily adherence record

## Minimal procedure
1) 设定本周三项基线目标
2) 拆分到每日最小动作
3) 执行并记录打卡
4) 识别连续偏离信号
5) 启动纠偏周策略
6) 周末复盘并更新阈值

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：生活健康基线 <输入>`
