# L0 Abstract - SOP Factory Production

Produce or upgrade repeatable SOPs into active status in one controlled cycle with strict, machine-verifiable release quality. 触发条件：task frequency is >= 3 per month AND outcome variance coefficient is <= 0.20; an active SOP has first-pass yield < 85% or rework rate > 20% in one review window。核心产出：SOP artifact triplet (`sop.md`, `scorecard.md`, `iteration-log.md`) with release decision; strict validation evidence and 1-3 rule updates for next cycle。
