# L1 Overview - 学习错题与薄弱点闭环

## When to use
- practice session finishes with mistakes logged
- same error type appears in 3 or more attempts

## Inputs
- Input 1: error entries
- Input 2: mastery threshold

## Outputs
- Output 1: corrective action plan per error category
- Output 2: updated mastery score and closed/open status

## Minimal procedure
1) 收集本轮错误样本
2) 按根因分类（概念/步骤/粗心）
3) 识别高频错误优先队列
4) 执行定向修复训练
5) 进行复测验证
6) 更新规则并沉淀模板

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：学习错题与薄弱点闭环 <输入>`
