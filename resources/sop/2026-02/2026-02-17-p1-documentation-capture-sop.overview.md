# L1 Overview - 文档沉淀与知识归档

## When to use
- 出现可复用结论或关键决策
- 同类问题重复出现 >= 2 次

## Inputs
- Input 1: 决策结论与背景
- Input 2: 证据链接和执行结果

## Outputs
- Output 1: 结构化知识条目
- Output 2: 可检索标签和引用链接

## Minimal procedure
1) 收集决策与证据来源
2) 按模板填写问题、结论、证据、行动
3) 添加标签与关联链接
4) 发布到统一目录
5) 加入检索索引
6) 在周复盘中检查复用情况

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：文档沉淀与知识归档 <输入>`
