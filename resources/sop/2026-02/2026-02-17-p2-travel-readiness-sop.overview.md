# L1 Overview - 旅行出差准备与执行

## When to use
- 确认出行计划
- 关键资料缺失或行程变更

## Inputs
- Input 1: 行程与预算信息
- Input 2: 证件和预订状态

## Outputs
- Output 1: 行前准备清单完成状态
- Output 2: 行中变更和应急记录

## Minimal procedure
1) 确认证件和关键预订
2) 完成行前清单打勾
3) 设置行程检查点和提醒
4) 准备应急联系人和替代路线
5) 行中记录变更并快速重排
6) 返程后复盘并更新清单

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：旅行出差准备与执行 <输入>`
