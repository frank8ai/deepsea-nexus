# L1 Overview - 家庭协同与分工沟通

## When to use
- 家庭固定任务超过5项或跨成员依赖明显
- 任务漏办连续2周 >= 2项

## Inputs
- Input 1: 家庭任务清单
- Input 2: 成员时间安排

## Outputs
- Output 1: 周分工与时间安排
- Output 2: 沟通纪要与待办

## Minimal procedure
1) 汇总本周家庭关键任务
2) 分配责任人和完成时间
3) 同步到共享日历
4) 执行中更新状态
5) 周沟通复核执行情况
6) 调整下周分工规则

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：家庭协同与分工沟通 <输入>`
