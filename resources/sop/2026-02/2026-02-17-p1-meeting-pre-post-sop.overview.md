# L1 Overview - 会议准备与行动闭环

## When to use
- 会议时长 >= 20 分钟或参会人 >= 3
- 连续2次会议行动项完成率 < 80%

## Inputs
- Input 1: 会议主题与背景
- Input 2: 参会人列表与时间窗口

## Outputs
- Output 1: 会议纪要与决策结论
- Output 2: 行动项清单（责任人、截止时间）

## Minimal procedure
1) 会前24小时明确会议目标与期望输出
2) 会前分发议程并收集补充议题
3) 会中按议程推进并控制时间
4) 会中实时标记行动项责任人与截止时间
5) 会后24小时发布纪要和行动项
6) 跟踪行动项并在下次会议复核

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：会议准备与行动闭环 <输入>`
