# L1 Overview - 学习迁移与应用实践

## When to use
- 完成一次学习会话或模块
- 连续2次学习后无迁移产出

## Inputs
- Input 1: 学习笔记和重点概念
- Input 2: 对应实践任务

## Outputs
- Output 1: 实践产出（题解或功能实现）
- Output 2: 迁移复盘记录

## Minimal procedure
1) 提取本次学习的3个关键概念
2) 为每个概念匹配一个实践任务
3) 执行最小实践并提交结果
4) 记录问题与卡点
5) 针对卡点进行定向补学
6) 完成迁移复盘并更新下次学习重点

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：学习迁移与应用实践 <输入>`
