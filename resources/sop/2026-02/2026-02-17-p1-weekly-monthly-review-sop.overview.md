# L1 Overview - 周月复盘与规则更新

## When to use
- 到达周末或月末复盘窗口
- 关键指标连续2周期反向变化

## Inputs
- Input 1: 周期指标数据
- Input 2: 执行日志和异常记录

## Outputs
- Output 1: 复盘结论和偏差原因
- Output 2: 1-3条规则更新

## Minimal procedure
1) 收集周期指标与目标值
2) 对比基线和当前值
3) 分析偏差根因
4) 制定1-3条规则更新
5) 发布规则并标注生效时间
6) 定义下周期验证计划

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：周月复盘与规则更新 <输入>`
