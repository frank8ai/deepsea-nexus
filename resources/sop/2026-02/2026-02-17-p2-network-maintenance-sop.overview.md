# L1 Overview - 人脉维护与周期触达

## When to use
- 关键联系人数量 >= 10
- 关键联系人连续30天未触达

## Inputs
- Input 1: 联系人名单和分层
- Input 2: 可用触达时间窗口

## Outputs
- Output 1: 本周期触达计划
- Output 2: 触达记录和下一步行动

## Minimal procedure
1) 更新联系人分层
2) 生成本周触达计划
3) 执行触达并记录反馈
4) 识别需跟进事项
5) 执行跟进并更新状态
6) 周期复盘调整分层与频率

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：人脉维护与周期触达 <输入>`
