# L1 Overview - 数字安全与备份权限管理

## When to use
- 存在关键数据或多账号系统
- 检测到异常登录或权限漂移

## Inputs
- Input 1: 资产与账号清单
- Input 2: 当前备份和权限状态

## Outputs
- Output 1: 安全检查报告
- Output 2: 整改任务与完成记录

## Minimal procedure
1) 盘点关键数据和账户
2) 执行备份并验证完整性
3) 审计权限与共享设置
4) 检查密码强度与复用
5) 处理异常项并复核
6) 输出本周期安全状态和下周期计划

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：数字安全与备份权限管理 <输入>`
