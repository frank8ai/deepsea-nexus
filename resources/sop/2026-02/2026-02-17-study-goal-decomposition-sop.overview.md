# L1 Overview - 学习目标拆解

## When to use
- a new learning objective is added
- weekly completion is below 75 percent

## Inputs
- Input 1: learning objective statement
- Input 2: available study hours

## Outputs
- Output 1: skill decomposition table
- Output 2: milestone schedule with checkpoints

## Minimal procedure
1) 定义学习终点能力
2) 拆分为技能树与子主题
3) 为每周设置里程碑
4) 定义检查点与达标阈值
5) 绑定学习资源与练习路径
6) 复核并冻结本阶段计划

## Quality gates
- Non-negotiables (legal/safety/security/data integrity) are explicitly checked.
- Objective is explicit and measurable.
- Outcome metric includes baseline and target delta.

## Invocation
`按SOP执行：学习目标拆解 <输入>`
